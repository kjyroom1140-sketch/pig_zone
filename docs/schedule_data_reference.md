## 8. 장소별 추가 가능한 작업 일정 요약

| 장소 | 선택 가능한 구분 | 해당 구분에서 선택 가능한 작업 대분류 |
|------|------------------|----------------------------------------|
| **자돈사** | 비번식돈, 시설 | 비번식돈: W01~W04, W07~W09 / 시설: W05~W10 |
| **자돈사(포유자돈)** | 비번식돈, 시설 | 위와 동일 |
| **분만사(모돈)** | 모돈, 자돈, 시설 | 모돈·자돈: W01~W04, W07~W09 / 시설: W05~W10 |
| **교배사** | 모돈, 시설 | 모돈: W01~W04, W07~W09 / 시설: W05~W10 |
| **임신돈사** | 모돈, 시설 | 위와 동일 |
| **육성돈사** | 비번식돈, 시설 | 자돈사와 동일 |
| **비육돈사** | 비번식돈, 시설 | 자돈사와 동일 |
| **웅돈사** | 옹돈, 시설 | 옹돈: W01~W04, W07~W09 / 시설: W05~W10 |
| **후보돈사** | 모돈, 시설 | 교배사와 동일 |

## 9. 테이블·시드 참고

| 테이블 | 시드 스크립트 |
|--------|----------------|
| schedule_divisions | seed_schedule_masters.js → DIVISIONS |
| schedule_bases | seed_schedule_masters.js → BASES |
| schedule_work_types | seed_schedule_masters.js → WORK_TYPES |
| schedule_work_detail_types | seed_schedule_masters.js → DETAILS_BY_WORK_TYPE |
| schedule_division_structures | seed_schedule_masters.js → mapping (structure_templates 존재 시) |

실제 DB는 `structure_templates` 행 기준으로 매핑되므로, 템플릿 name이 위와 다르면 schedule_division_structures 결과도 달라질 수 있습니다.
