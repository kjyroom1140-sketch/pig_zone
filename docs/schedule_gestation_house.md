# 임신사 돼지 관리 일정 스케줄

임신사(임신돈사)에서 **전입일 기준**으로 하는 돼지(모돈) 관리 일정을 단계별로 정리한 문서입니다.

---

## 1. 기준: 전입일(임신사 입식일)

- **전입일 0일** = 임신 확인 후 임신사로 이동한 날.
- 임신 기간 약 **114~115일** 중, 임신사 체류는 보통 **전입 0~90일** 구간(임신 초·중·후기)이며, 이후 **분만사 이동**.

---

## 2. 일정 스케줄 요약 (전입일 기준)

| 전입 일수 | 구간       | 작업 유형 | 작업 내용 |
|-----------|------------|-----------|-----------|
| **0**     | 전입 당일  | 관리      | 전입일 기록·군 배치 |
| **0~7**   | 적응기     | 관찰      | 전입 후 스트레스·적응 관찰 |
| **0~14**  | 초기       | 관찰      | 건강 상태·사료 섭취 관찰 |
| **0~90**  | 전 기간    | 환경      | 환경·위생 관리 |
| **0~90**  | 전 기간    | 사양      | 임신 단계별 사료 급여 |
| **30**    | 임신 초기  | 예방접종  | 임신 중 1차 접종 |
| **60**    | 임신 중기  | 예방접종  | 임신 중 2차 접종 |
| **80~90** | 임신 후기  | 관리      | 분만사 전입 준비·체중 점검 |
| **85~90** | 출산 직전  | 이동      | 분만사 이동 |

---

## 3. 단계별 상세

### 3.1 전입 직후 (0~7일)

- **관리**: 전입일 기록, 검역 구역 또는 군 배치.
- **관찰**: 이동 스트레스·적응, 식욕·건강 상태 확인.

### 3.2 임신 초기 (0~30일)

- **환경**: 온·습도·환기, 돈방 위생.
- **사양**: 임신 초기 사료(에너지·영양 균형).
- **관찰**: 발정·이상 발정 여부, 건강·사료 섭취량.
- **예방접종**: 30일 전후 1차 접종(농장 계획에 맞게).

### 3.3 임신 중기 (30~60일)

- **환경·위생**: 지속 관리.
- **사양**: 임신 중기 사료로 전환·급여량 조절.
- **예방접종**: 60일 전후 2차 접종.

### 3.4 임신 후기 (60~90일)

- **사양**: 임신 후기·분만 전 사료(포유 준비).
- **관리**: 체중·비만도 점검, 분만사 전입 일정 확인.
- **이동**: 전입 85~90일 전후 분만사 이동.

### 3.5 시설(임신사) 일정

- **매일**: 환경 점검(온도·습도·환기).
- **주 1회**: 위생·소독.
- **전입 시**: 전입 시 시설 점검·준비.
- **종료 시**: 종료 후 세척·소독.

---

## 4. DB 반영 현황

- **돼지(pig) 일정**: `schedule_items` — 대상장소 = 임신사, 기준 = 전입일.
- **시설(facility) 일정**: 매일/주 1회/전입·종료 시 시설 관리.
- 신규·보완 항목은 `scripts/seed_schedule_gestation_house.js` 로 추가 가능.

---

## 5. 빠른 필터에서 보기

관리자 **일정 항목** 화면에서 **빠른 필터**는 현재 교배사·분만사·자돈사만 지원합니다.  
임신사만 보려면 **대상장소** 드롭다운에서 "임신사"를 선택 후 **조회**하면 됩니다.
